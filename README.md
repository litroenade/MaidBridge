# MaidBridge

MaidBridge 是给 Touhou Little Maid 用的外部 AI 桥接 Mod。它在 Minecraft 里开一个 WebSocket 服务，允许外部agent接管女仆聊天、读取女仆状态，并在允许时回写动作。

## 使用说明

```
/give @p maidbridge:maid_uuid_probe
```

通过指令获取女仆uuid

## 推荐已有外部agent接入

- [MaiBot](https://github.com/litroenade/maibot-maid-adapter)
